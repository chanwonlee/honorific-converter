# 한국어 텍스트 스타일 변환 이메일 시스템

- 배포 URL:

## 프로젝트 소개

- 한국어가 서툰 외국인 대학생들과 메일 보내기가 어려운 대학생들을 위한 메일 전송 프로그램입니다.
- 한국어와 중국어를 선택할 수 있으며, 작성한 문장을 경어체로 자동 전환해줍니다.
- 메일 전송 링크, 영문 변환, 복사 버튼등 다양한 편의 기능을 제공합니다.
